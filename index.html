What is Neo

Neo is a machine learning library written in JavaScript, which is built on AgentSmith. Neo provides efficient, simple, and powerful tools for data analysis.

Getting Started

In order to load codes in appropriate order while meeting dependency, assembling all codes to one file is necessary. It can be done by running following command. 
$ cd your_repo_root/neo
$ python compile.py
After that, neo.js is generated. When Neo is used, this file should be read. You can understand how to use Neo through demonstration. The only thing you have to do is to select method and run it. Data and parameters of model can be changed. In this demonstration, Trinity, drawing library build on AgentSmith, is used to plot points and draw curves. For more information, see Neo Documentation page.
